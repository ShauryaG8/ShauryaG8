### IoT-Enabled Smart Helmet for Real-Time Environment Monitoring

The helmet continuously monitors hazardous environmental conditions using temperature, humidity, MQ7 (CO gas), and MQ35 (ammonia gas) sensors, while an IR sensor detects nearby obstacles. Key features include:
• Real-time alerts via a built-in display for immediate hazard warnings.
• Battery-powered portability for on-site usability.
• Cloud-based data logging to track historical trends and ensure compliance with safety protocols.
This project aimed to mitigate health risks in construction zones by providing actionable insights to workers and supervisors, reducing accidents caused by toxic exposure or environmental extremes.
